🧭 Study Guidelines & Strategy ▼
🎯 Role Focus

Target Platform Engineer / Senior DevOps / Cloud Infrastructure Engineer. Emphasize building reusable infrastructure, automation, CI/CD platforms, and developer enablement — not incident-heavy SRE.

⏱ 1-Hour Session Protocol
  • ✅ Pick ONE card from the "To Study" column
  • ✅ Set a timer for 60 minutes
  • ✅ Complete the core task + take 1 note
  • ✅ Move card to "Done" and log progress
✍️ Resume Bullet Rules
  • Use Built / Automated / Designed / Standardized over Managed / Supported
  • Format: Action + System + Impact
  • Add metrics: frequency, time saved, environments, failure reduction
  • Lead with 6+ years cloud engineering & Terraform-driven AWS
🧠 Platform Mindset
  • Build once, reuse across teams
  • Standardize environments and release paths
  • Reduce manual work and ticket dependency
  • Improve developer experience
  • Create safe defaults and templates
📬 Weekly Outreach Target
  • 📄 15–20 tailored applications
  • 💬 5 recruiter messages
  • 🤝 5 referral requests
  • 🔁 2 follow-ups on prior applications
🚫 Roles to Avoid

Skip roles heavy in: on-call, L3 support, NOC, ops-heavy maintenance. Look for: Terraform, AWS, IaC, automation, platform, CI/CD, Kubernetes, internal tooling, developer experience.
