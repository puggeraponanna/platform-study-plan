# 📚 Daily Study Planner — Kanban

A Notion-style drag-and-drop Kanban board for organizing daily study sessions.

## Features
- 🗂 4 columns: To Study · In Progress · Review · Done
- 🖱 Drag & drop cards between columns
- 🔗 Learning resource links per card
- 🔴🟡🟢 Priority tags & time estimates
- 📊 Progress bar
- 💾 Auto-saves to localStorage
